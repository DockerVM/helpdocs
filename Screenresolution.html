Screen resolution

The configuration panel ›› Video ››

Screen resolution: Affects the resolutions detected by the application. There is a choice of the following values:


  • SVGA modes: the classic 4:3 screen resolutions starting from 320x200 up to 1280x800
  • HDMI modes: the typical 16:9 resolutions from 640x360 up to 1980x1080
  • Monitor native modes: whatever returned from the video card


Notice that 320x200 resolution is typically no longer supported, but it is not a problem to emulate it in window and this resolution is necessary to run some older games.


Limit resolution: Disables any resolution higher than the selected value. By default, the choice is "unlimited", that means no resolution is disabled.
