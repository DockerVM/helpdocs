The Interface ››


Run: Starts the currently selected application

Modify: Opens the configuration panel to set/change the selected program settings

Delete: Deletes the selected application entry (asking a Yes/No confirmation)

Add: Inserts a new application entry in the list. The configuration panel is opened to let you define the initial settings.

Explore: Opens Microsoft Explorer to the folder where the application is located. This is a shortcut to something  usually useful.

Log: See Log.
